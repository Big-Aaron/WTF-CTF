# WTF-CTF
CTF（Capture The Flag，夺旗赛）是一种网络安全竞赛，源于黑客和网络安全社区。CTF比赛通常由一系列安全挑战组成，参赛者需要发挥他们的技术、创造力和解决问题的能力，以找出并利用各种网络安全漏洞来完成任务。比赛的目的是让参赛者在模拟的真实场景中学习和练习网络安全技能。
